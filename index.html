Our Content

Package 1: CLOSE IT DOWN, HIT EM' BACK

Description

This packages contains two programs that serve two different function. One for defense against hackers and the other allows you to go on the offensive!

Program 1: MONITOR AND CLOSE IT DOWN

Description: This script allows you to monitor the number of open processes of Notepad. Giving you an exact number of how many tabs are open. With a built in kill switch, all these tabs can be closed with one simple Y/N input. Which can aid at preventing possible lag or your system from becoming overwhealmed with requests.

Program 1: HIT EM' BACK

Description: This scripts hits those nasty hackers where it hurts. On the surface this program seems innocent enough, it generates a random letter and gives them a tongue twister with that letter! How fun! But in reality, it actually deletes all files that contain said letter in their name...good luck getting those back!
